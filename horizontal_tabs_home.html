Course Description

Course description (TL;DR)

  • Module 1 - OMGs! Self - Outcome, Metacogition, Goals

    • Self-Theory of accomplishment (i.e, Growth mindset, Positive psychology, Dark psychology and More)
    • iSMART goals create learning goals for a better you and life

      • Self-Regulated learning (SRL) and Self-Directed Learning (SDL)

    • Program Learning Outcomes (PLOs) and Course Learning Outcomes (CLOs)

  • Module 2 - ACE - Assessment Competency and Evaluation

    • Test-enhanced learning

    • Desirable difficulty

    • Feedback and Reflection

  • Module 3 - Learning - State of Art in Effective learning strategies

    • Spacing

    • Interleaving

    • Deliberate practice
